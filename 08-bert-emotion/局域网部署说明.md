# 投满分系统 - 局域网部署说明

## 📋 修改内容

已将系统配置为支持局域网访问，其他电脑可以通过你的 IP 地址 `192.168.12.63` 访问系统。

### 🔧 修改的文件

1. **`api_flask_server.py`**
   - 将监听地址从 `192.168.12.63` 改为 `0.0.0.0`（监听所有网络接口）
   - 添加了访问地址提示

2. **`app_streamlit_run.py`**
   - 添加 `--server.address 0.0.0.0` 参数
   - 指定端口为 `8501`
   - 添加了访问地址提示

3. **新增 `start_all.bat`** - Windows 一键启动脚本

4. **新增 `start_all.py`** - Python 一键启动脚本

---

## 🚀 启动方式

### 方式一：一键启动（推荐）

#### Windows 用户：
```bash
# 双击运行
start_all.bat
```

#### Python 用户：
```bash
python start_all.py
```

---

### 方式二：分别启动

#### 1. 启动 Flask API 服务
```bash
python api_flask_server.py
```
- API 地址：http://192.168.12.63:80
- 健康检查：http://192.168.12.63:80/health

#### 2. 启动 Streamlit 前端
```bash
python app_streamlit_run.py
```
或直接使用 Streamlit 命令：
```bash
streamlit run app_streamlit.py --server.address 0.0.0.0 --server.port 8501
```
- 前端地址：http://192.168.12.63:8501

## 🔒 防火墙设置

如果局域网其他电脑无法访问，请检查 Windows 防火墙设置：

1. **允许 Python 通过防火墙**
   - 打开「Windows Defender 防火墙」
   - 点击「允许应用通过防火墙」
   - 找到 Python，勾选「专用」和「公用」网络

2. **或者临时关闭防火墙测试**（仅用于测试）

---

## 📝 注意事项

1. **确保所有设备在同一局域网**
2. **IP 地址确认**：如果 `192.168.12.63` 发生变化，请修改相关配置
3. **端口占用**：确保 `80` 和 `8501` 端口未被占用
4. **Streamlit 默认 API 地址**：在前端页面选择「通过API调用」模式时，默认地址已设置为 `http://192.168.12.63:80/predict/all`
